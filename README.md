# domod
Platform independent domotica data gatherer. Known supported hardware:
 - Raspberry Pi 1B+

# Installing

```
sudo su -
cd /path/to/where/you/want/store/domod
git clone https://github.com/Mausy5043/domod.git
cd domod
./install.sh
./update.sh
```
